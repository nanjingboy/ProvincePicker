# ProvincePicker

Chinese province picker library for Android.

# Screenshot

![ProvincePicker Screenshot](Screenshot.gif)

# Setup

* In your `build.gradle` :

```gradle
repositories {
    maven { url "https://jitpack.io" }
}

dependencies {
    compile 'com.github.nanjingboy:ProvincePicker:1.0.1'
}
```
